# Install as a plugin
```
/plugin marketplace add IFC-UIDAHO/Cambium_AI
/plugin install cambium-institute
```
The plugin bundles the 45 agents in `.claude/agents/`. After install, open `dashboard.html` and use the
command phrases (see the top-level README). Prefer copy-paste? Just clone the repo and copy
`.claude/agents/` into your project.
