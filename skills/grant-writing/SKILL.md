# Grant writing — win on the funder's own criteria

Reviewers score against the solicitation's criteria, not against how nice the prose is. **Extract the
criteria first, then write every section to earn points on them.**

## Always start from the solicitation
Pull, verbatim where possible: eligibility, page/format limits, review criteria + weights, required
documents, budget caps (incl. indirect/F&A), and deadlines. Mirror the funder's language. If a fact
isn't in the solicitation or the PI's inputs, ask — don't invent.

## Specific Aims (the make-or-break page)
- **Hook** (problem + gap), **long-term goal + objective**, **central hypothesis** + its basis.
- **2–4 aims**, each a testable objective (not a method list); independent enough that one failing
  doesn't sink the rest.
- **Payoff**: what changes if it succeeds. One page, skimmable.

## Section map (adapt to the funder)
| Section | Job |
|---|---|
| Significance | the problem matters; the gap is real and important |
| Innovation | what's genuinely new vs prior art (be honest, specific) |
| Approach | aims → methods → expected outcomes → **pitfalls & alternatives** → timeline |
| Rigor & feasibility | why this team/preliminary data make it doable |
| Broader impacts / outcomes | NSF broader impacts, NIH significance, USDA stakeholder relevance |

## Funder accents
- **NSF**: Intellectual Merit + **Broader Impacts** as co-equal; 15-page limit typical.
- **NIH**: Significance · Innovation · Approach · Investigators · Environment; Aims page is king.
- **USDA-AFRI**: relevance to program priorities + stakeholder/extension impact.

## Map the draft back to criteria
Build a table: each review criterion → where the proposal addresses it → strength. Fix every gap before
submission. Hand to the referee agent for a scored mock review.

## Guardrails
- Never fabricate preliminary data, citations, or collaborator commitments — flag what's needed instead.
- Keep claims within their evidence tier (proposed ≠ proven). The budget/compliance pieces are
  grants-compliance + budget-officer's job; coordinate, don't guess numbers.
- Respect every format/page rule — non-compliance gets desk-rejected regardless of quality.
